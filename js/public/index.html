DearReader API - Local Web Content Extractor

Convert any URL to LLM-friendly content

📚 Try These Examples

  • Basic: example.com
  • Basic: news.ycombinator.com
  • Basic: github.com/jina-ai/reader
  • Queue Monitor

🔧 Usage

curl -H "Accept: application/json" "/https://example.com"
curl "/https://example.com"
curl -H "X-Respond-With: screenshot" "/https://example.com"

🌐 Response Formats

  • Default: Clean markdown content
  • JSON: Complete metadata with links, images, and content
  • HTML: Cleaned HTML content
  • Text: Plain text extraction
  • Screenshot/Pageshot: URL to saved image
